# SmartOffice AI — Site Institucional

Primeira versão do site institucional da SmartOffice AI, construída com Next.js, TypeScript e CSS responsivo.

## Como instalar

Substitua os arquivos equivalentes no projeto local:

- `src/app/page.tsx`
- `src/app/layout.tsx`
- `src/app/globals.css`
- `src/app/icon.svg`

Depois execute:

```bash
npm run dev
```

Acesse `http://localhost:3000`.

## Publicar

```bash
git add .
git commit -m "Create SmartOffice AI institutional website"
git push origin main
```

A Vercel fará o deploy automaticamente.
